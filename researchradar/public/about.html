About ResearchRadar

A free academic discovery tool built to help students and researchers find relevant papers without paywalls or sign-ups.

The Problem We Solve

University students and independent researchers face a frustrating reality: the best academic discovery tools are paywalled, require institutional accounts, or simply don't have APIs. Google Scholar has no public API. Most alternatives are proprietary.

ResearchRadar changes that. It's built entirely on free, open APIs — no account required, no subscription, no limits imposed on you.

Powered By Open APIs

📚

OpenAlex API

openalex.org ↗

The world's largest open catalog of scholarly works. Indexes over 250 million papers, authors, institutions, and concepts — fully free, no API key required.

  • ✓ 250M+ works indexed
  • ✓ No authentication needed
  • ✓ Abstract & citation data
  • ✓ Open Access PDF links
🌐

REST Countries API

restcountries.com ↗

A free, public API providing data about every country in the world. Used by ResearchRadar to populate the author country filter dropdown with proper country names and codes.

  • ✓ 250 countries & territories
  • ✓ No API key required
  • ✓ ISO country codes (cca2)
  • ✓ Always up to date

How It Works

01

Enter a Query

Type any research topic, author name, institution, or DOI into the search bar on the Discover page.

02

Apply Filters

Narrow results by year range, open access status, author country, or minimum citation count.

03

Explore Papers

Browse results in list or grid view. Click any paper to see its full abstract, author details, and citation stats.

04
